前置说明

因为使用了 layui 作为主题框架，这里使用到了 layer 和 layui 的样式， 表单内容摒弃了 layui 中的 form.render 渲染形式， 使用 vue 和 layui 的样式进行定义，内容使用 vue 的 MVVM 进行交互， 所有的内容不定义到 DOM 中，例如表单的 input 中 name 属性不进行渲染； hidden 属性的 input 不进行渲染，直接记录在内存中等等。
